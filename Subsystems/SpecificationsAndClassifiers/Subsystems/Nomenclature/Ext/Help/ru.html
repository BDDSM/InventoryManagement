Настройка ведения номенклатуры

При работе со списком номенклатуры необходимо произвести первоначальное заполнение и поддерживать в актуальном состоянии информацию в справочниках: Виды номенклатуры, Единицы измерения.

Формирование и поддержка упаковок номенклатуры

Информация о всех видах упаковок, которые могут использоваться на предприятии, вводится в справочник Единицы измерения. Для каждой единицы измерения, введенной в справочник, заполняется информация о коде единицы измерения, кратком и полном наименовании единицы измерения, а также международное сокращение единицы измерения.
Все реквизиты следует заполнять в соответствии с Общероссийским классификатором единиц измерения (ОКЕИ).

Формирование и поддержка видов номенклатуры

Информация о всех видах номенклатуры, используемых на предприятии, заполняется в справочнике "Виды номенклатуры". Справочник "Виды номенклатуры" - это классификатор, предназначенный для объединения номенклатуры по общим признакам учета. В конфигурации предусмотрены варианты ведения дополнительного учета номенклатуры в зависимости от ее вида - например, учет по дополнительным характеристикам и учет упаковок номенклатуры. Используя этот классификатор, можно задать произвольные дополнительные сведения для номенклатуры и характеристик, относящихся к определенному виду номенклатуры.

Информация о новом виде номенклатуры добавляется в справочник Виды номенклатуры. В форме вида номенклатуры вводится основная информация о виде номенклатуры:

  • Наименование. Название вида номенклатуры.
  • Группа. Группа, в которую входит данный вид номенклатуры. Данная возможность используется в том случае, если на предприятии используется много различных видов номенклатуры, которые необходимо объединять дополнительно в группы.
  • Тип номенклатуры. Указывается тип, к которому относится номенклатура данного вида. Можно выбрать только одно из двух значений: Товар или Услуга.

Выбирается вариант использования характеристик для данного вида номенклатуры. (Использовать характеристики). Данную возможность можно использовать только в том случае, если включена функциональная опция Использовать характеристики номенклатуры.
Можно выбрать следующие значения:

  • Не использовать. Для данного вида номенклатуры не ведется учет в разрезе характеристик.
  • Использовать общие для вида номенклатуры. Назначается общий список характеристик, который используется всеми позициями номенклатуры, которые принадлежат данному виду.
  • Использовать индивидуальные для номенклатуры. Для каждой позиции номенклатуры назначается индивидуальный список характеристик.

Вводится информация о варианте использования упаковок для данного вида номенклатуры (Использовать упаковки). Данная возможность доступна только в том случае, если включена функциональная опция Использовать упаковки номенклатуры. Можно выбрать следующие значения:

  • Не использовать. Для данного вида номенклатуры не используются упаковки.
  • Использовать общие для вида номенклатуры. Назначается общий список упаковок, который используется всеми позициями номенклатуры, принадлежащими данному виду.
  • Использовать индивидуальные для номенклатуры. Для каждой позиции номенклатуры назначается индивидуальный список упаковок.

При записи вида номенклатуры автоматически создается набор свойств с названием, соответствующим названию вида номенклатуры. В список наборов свойств необходимо добавить те дополнительные реквизиты и дополнительные сведения, которые будут использоваться для позиций номенклатуры, принадлежащим данному виду.

Для тех видов номенклатуры, для которых установлен флаг Использовать общие для вида номенклатуры, можно указать список характеристик, который будет использоваться для всех тех номенклатурных позиций, которые относятся к данному виду. Для ввода списка характеристик, которые применяются для данного вида номенклатуры, после записи информации о виде номенклатуры, необходимо выбрать пункт Характеристики номенклатуры в панели навигации формы. В список добавляются все те характеристики, которые используются для данного вида номенклатуры.


Пример. Товары вида Обувь характеризуется размером и цветом модели обуви. В список характеристик добавляется все возможные сочетания цвета и размера обуви: Размер 45, Цвет белый; Размер 37, Цвет белый; Размер 37, Цвет черный и т.д.
При вводе информации о товаре, который принадлежит данному виду, его характеристику можно будет выбрать из списка тех доступных характеристик, которые определены для данного вида.

Для тех видов номенклатуры, для которых выбран вариант использования упаковок Использовать общие для вида номенклатуры, можно указать список упаковок, который будет использоваться для всех тех номенклатурных позиций, которые относятся к данному виду.

Для ввода списка упаковок, которые применяются для данного вида номенклатуры, после записи информации о виде номенклатуры, необходимо выбрать пункт Упаковки номенклатуры в панели навигации формы. В список добавляются все те упаковки, которые используются при покупке или продаже товаров данного вида. При заполнении используется общий список упаковок (справочник Единицы измерения). Для каждой упаковки вводится информация о названии упаковки и коэффициент (количество товаров в упаковке)

Пример. Товары вида Кондитерские изделия могут отгружаться как в коробках, так и в ящиках (25 коробок). Для этого вида номенклатуры определяется упаковка ящик с коэффициентом, равным 25.
При вводе информации о товаре, который принадлежит данному виду, информацию об упаковке можно будет выбрать из того списка доступных упаковок, которые определены для данного вида номенклатуры.

Ведение списка номенклатуры

Информация о позициях номенклатуры (товарах и услугах) вводится в справочник Номенклатура.


Просмотр списка позиций номенклатуры

Для просмотра списка позиций номенклатуры, которые используются на предприятии, необходимо открыть справочник Номенклатура. В справочнике показывается общий список номенклатуры. Список номенклатуры может быть представлен в виде списка, в виде иерархического списка (с указанием групп номенклатуры), а также в виде дерева. Порядок представления данных в списке определяется установленным режимом просмотра (Все действия - Режим просмотра).

Ввод информации о новой позиции номенклатуры

В карточку новой позиции номенклатуры вводится следующая информация.

  • Наименование. Название товара или услуги, которое используется для быстрой визуальной идентификации номенклатуры в процессе работы с программой.
  • Полное наименование. Наименование товара и услуги, которое будет печататься во всех документах
  • Артикул. Артикул позиции номенклатуры.
  • Единица измерения. Единица измерения, в которой ведется учет складских остатков товаров и указывается количество номенклатуры в одной упаковке. Для заполнения единицы измерения используется справочник Классификатор единиц измерения.
  • Группа. Группа, в которую входит данная позиция номенклатуры в соответствии с классификацией (иерархией) товаров в справочнике Номенклатура.
  • Вид номенклатуры. Указывается вид, к которому следует отнести данную позицию номенклатуры. При выборе вида номенклатуры справочно указываются те параметры, которые характеризуют данный вид номенклатуры: тип номенклатуры (товар или услуга), использование характеристик, использование упаковок.
  • Ценовая группа. Ценовая группа, в которую входит позиция номенклатуры, в соответствии с классификацией товаров по ценовым характеристикам.
  • Использовать серийные номера. Если этот флаг установлен, то для данной позиции номенклатуры можно вести учет в разрезе серийных номеров.
  • Вести учет по ГТД. Признак того, что для элемента номенклатуры необходимо вести учет по номерам ГТД.

После ввода позиции номенклатуры необходимо нажать на кнопку Записать. После записи позиции номенклатуры для нее могут быть добавлены дополнительные реквизиты и дополнительные свойства, если они определены для того вида номенклатуры, к которому относится данная позиция (определен набор свойств для вида номенклатуры).

Для позиции номенклатуры можно ввести список тех индивидуальных характеристик, которые назначаются для конкретной позиции номенклатуры. Для ввода дополнительных характеристик используется справочник Характеристики, который показывается в панели навигации формы. Информация в этом справочнике заполняется пользователем только в том случае, если для вида номенклатуры, к которому принадлежит данная позиция, предусмотрено ведение индивидуального списка характеристик. Если для вида номенклатуры используется общий список характеристик, то он будет показан для просмотра, без возможности его дополнения. Ввод новых характеристик в этом случае производится в справочнике Виды номенклатуры.

Для позиции номенклатуры можно ввести список тех индивидуальных упаковок, которые назначаются для конкретной позиции номенклатуры. Для ввода упаковок используется справочник Упаковки, который показывается в панели навигации формы. Информация в этом справочнике заполняется пользователем только в том случае, если для вида номенклатуры, к которому принадлежит данная позиция, предусмотрено ведение индивидуального списка упаковок. Если для вида номенклатуры используется общий список упаковок, то он будет показан для просмотра, без возможности его дополнения. Ввод новых упаковок в этом случае производится в справочнике Виды номенклатуры.

Ввод дополнительных реквизитов производится на закладке Доп.реквизиты. Информация на этой закладке заполняется в соответствии с теми дополнительными реквизитами, которые были заданы в наборе свойств для вида номенклатуры, к которому принадлежит данная позиция номенклатуры. Необходимо ввести значения этих дополнительных реквизитов. Для ввода значений могут использоваться различные справочники, также это может быть текстовая информация, число, дата и т.д. Это зависит от того, как был определен дополнительный реквизит в наборе свойств.

Дополнительные сведения по номенклатурной позиции вводятся в отдельном диалоговом окне, которое открывается при нажатии на соответствующую пиктограмму или при выборе действия Дополнительные сведения. В списке показываются все те дополнительные сведения, которые были определены в наборе свойств для того вида номенклатуры, к которому принадлежит данная позиция номенклатуры. Пользователь должен ввести значения этих дополнительных сведений. В качестве значений могут использоваться различные виды значений: число, дата, текстовая строка, различные справочники. Это определяется тем какой тип значения задан для дополнительного свойства в наборе свойств для данного вида номенклатуры.

Для позиции номенклатуры можно определить список тех серийных номеров, которые могут быть использованы для данного товара. Список серийных номеров добавляется в справочник Серийные номера, который расположен на панели навигации формы. Справочник появляется в списке в том случае, если для позиции номенклатуры ведется учет по серийным номерам (установлен флаг Использовать серийные номера). Информация о серийном номере может быть добавлена в справочник в момент оформления поступления товаров.

Для импортных товаров можно ввести информацию о номере грузовой таможенной декларации и стране происхождения, из которой поступил импортный товар. В дальнейшем эта информация будет использоваться при оформлении счетов-фактур. Информация вводится в справочник Номера ГТД. , который расположен на панели навигации формы. Справочник появляется в списке в том случае, если для позиции номенклатуры ведется учет по номерам ГТД (установлен флаг Вести учет по ГТД). Номер ГТД заполняется в виде текстовой строки, а информация о стране происхождения выбирается из соответствующего справочника. Информацию о номере ГТД и стране происхождения можно заполнить и в момент оформления документа поступления товаров.

Для одного товара можно добавить несколько штрихкодов. Штрихкод товара детализируется до уровня характеристик и упаковок товара. То есть можно назначить различные штрихкоды для каждой упаковки, в которой поставляется и продается товар, а также для каждой из характеристик, которые назначены для товара. Информация о штрихкодах товара вводится в справочник Штрихкоды, , который расположен на панели навигации формы.

Для номенклатурной позиции указывается та ячейка, в которой она находится на конкретном складе. Информация вводится в регистр сведений Размещение номенклатуры по складским ячейкам.

Просмотр дополнительной информации о товаре

Из формы номенклатуры можно посмотреть дополнительную информацию о ценах, которые назначены данной позиции номенклатуры. Для просмотра используются соответствующие справочники, которые расположены на панели навигации формы.

  • Цены номенклатуры. Показывается список цен, зарегистрированных для данной позиции номенклатуры, документом Установка цен номенклатуры.
  • Цены партнеров. Список цен партнеров (конкурентов, поставщиков и т.д.), зарегистрированных для данной позиции номенклатуры. Информация об этих ценах регистрируется при оформлении документа Установка цен номенклатуры партнера.
