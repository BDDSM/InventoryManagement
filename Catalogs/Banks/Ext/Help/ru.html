Справочник "Банки"

Справочник содержит сведения о банках, в которых у организаций предприятия  и ее партнеров (контрагентов) имеются расчетные счета.  Информация о банке может быть заполнена двумя способами.

1-й способ. Информация о банке заполняется вручную.

Нажмите клавишу Insert или используйте команду  Создать.

В новой карточке банка заполните следующие поля.

  • Группа - группа, в которую входит данный банк.
  • Наименование - наименование банка.
  • Корреспондентский счет - корреспонденский счет банка.
  • БИК - банковский идентификационный код.
  • Город - данные о местонахождении банка.
  • Адрес - почтовый адрес банка.
  • Телефоны - телефоны для связи с сотрудниками банка.

2-й способ. Информация о банках заполняется с помощью общероссийского классификатора банков (ОКБ).

  • Находясь в списке банков нажмите кнопку Добавить из классификатора.
  • Откроется форма для выбора банков из общероссийского классификатора банков.
  • Для отбора банков из классификатора можно воспользоваться различными вариантами отбора: по наименованию банка, по городу, в котором находится банк, по корреспондентскому счету или по БИК.  
    • Установите нужные параметры отбора и нажмите на кнопку Подобрать банки.
    • Отметьте в списке те банки, которые необходимо добавить в справочник.
    • Нажмите кнопку Ввод (ENTER).  
    • Информация о банках из классификатора будет заполнена в справочнике Банки.
